Wireshark: Open source network capture and analysis.
Captures data packets traveling through a network in real time. Used to look at packet headers and payloads, source and destination addresses, application-level data. 
Widely used in (network) forensics, tracing through an attack or protecting against unencrypted protocols that can be leveraged.

CyberChef: web app, can decode, encode, and transform data using various methods and tools. 

Volatility: Used to analyse RAM dumps, needed in malware analysis or forensics.
